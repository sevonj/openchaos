# RULES.md

This file cannot be modified or deleted. PRs attempting to do so will fail CI.

## Merge Rules

- PRs are ranked by community vote
- Ties go to the more recent PR
- Highest-ranked mergeable PR wins

## Exceptions

The maintainer may reject PRs containing code designed to harm users or systems.

## Immutability

This file and its CI protections cannot be removed by vote.
Everything else can.
